# AI Data Policy Compliance Agent

An AI-driven solution to ensure data policy compliance, automatically evaluating and reviewing documents against organizational and regulatory policies.

## Features
- **Frontend**: A user-friendly interface for document upload and policy management.
- **Backend**: An intelligent backend processing engine.

## Directory Structure
- `frontend/`: Contains the frontend application.
- `backend/`: Contains the backend logic and AI models.

## Getting Started
Please refer to the respective `README.md` files in the `frontend` and `backend` directories for specific setup instructions.
